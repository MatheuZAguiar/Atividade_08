# EntregaSpring
Esta é uma aplicação Spring Boot para calcular Média , Desvio Padrão , Mediana e Quantidade de Dados recebidos de um conjunto de valores.

## Configuração
1 - Clone este repositório para seu ambiente local.

2 - Certifique-se de ter o Java 8 ou superior instalado.

3 - Abra o projeto em sua IDE preferida.

4 - Inicie o aplicativo Spring Boot.

## Uso
### Adicionar Valor

Endpoint: `POST/valores`

Para adicionar um valor , faça uma solicitação POST usando o seguinte formato JSON:

```json
{
  "valor": 10.5
}
